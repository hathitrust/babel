HathiTrust Data API key request

The HathiTrust Data API provides access to data and metadata resources. URLs for restricted data or higher data rates must include an access key and be signed with a secret key. An access key and signature are not required to request metadata resources or unrestricted data resources.

Consult the HathiTrust Data API documentation to see an example program to produce a signed Data API resource request. Copy and edit it to use the keys you request here as a starting point to access the Data API using signed URLs.
