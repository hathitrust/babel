HathiTrust Data API key request

The HathiTrust Data API provides access to data and metadata resources. Data API request URLs for data resources, such as page images and OCR files, must include an access key and be signed with a secret key. An access key and signature are not required to request metadata resources.

For more information on how to use the Data API to request HathiTrust resources please consult the HathiTrust Data API documentation. Contact feedback@issues.hathitrust.org regarding enhanced access privileges.

Instructions

  • When we receive your information, we'll send email containing a link to the registration page. You may need to allow feedback@issues.hathitrust.org to pass your spam filter.
  • Follow the registration link in the email to view your keys. Copy them to a secure location and treat them the same as passwords.
  • The registration link is time-sensitive and one-time-only. It expires after the first visit and absolutely after 24 hours.
  • There is a limit of five keys per email address. Keys granting enhanced access expire six months from the date of issue.
    • Please contact us at feedback@issues.hathitrust.org if you need assistance.
